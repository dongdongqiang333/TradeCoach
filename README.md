## 产品是什么

**交易教练** 是一个基于 OKX Agent Trade Kit 的 AI 交易行为分析工具。

大多数散户亏钱不是因为不懂行情，而是因为**交易行为本身出了问题**：
- 追涨杀跌，明知道不该做还是手滑
- 止损设了不执行，一路扛到爆仓
- 亏了之后报复性加仓，越亏越多
- 不知道自己哪个时间段、哪类交易最容易亏

交易教练通过 OKX ATK 实时读取你的交易数据，像健身 App 一样帮你**量化交易习惯、追踪进步、建立纪律**。

## 核心功能

### 01 · 今日交易健康度评分

每天收盘后，用 OKX ATK 读取当日所有交易记录，从五个维度综合打分（0-100）：

| 维度 | 评分逻辑 | 调用工具 |
|------|---------|---------|
| 风险控制 | 杠杆率是否合理，仓位是否分散 | `getPositions` |
| 止盈执行 | 设定目标价后是否真的止盈 | `getTradeHistory` |
| 情绪稳定 | 是否在市场极端情绪时追涨杀跌 | `getTrades` + OpenClaw |
| 仓位管理 | 单笔仓位是否超过总资金10% | `getAccountStats` |
| 策略执行 | 实际操作和预设策略的吻合度 | `getTrades` |

**评分等级：**
- 🟢 80-100 · 优秀交易者
- 🟡 60-79  · 良好，继续保持
- 🔴 0-59   · 需要注意，今日交易有明显问题

---

### 02 · 下单前 AI 风险评分

在你点击下单的瞬间，交易教练通过 OKX ATK 自动完成以下检查，并在 **3秒内** 返回风险评分：

```
用户发起下单
    ↓
ATK · getRiskLevel     → 账户当前风险等级
ATK · getMarketData    → 当前趋势 / 资金费率
ATK · getPositions     → 当前仓位集中度
OpenClaw · 情绪采集    → 市场情绪是否极端
    ↓
AI 综合评分（0-100，分数越低越安全）
    ↓
< 30分  → 绿色放行，直接执行
30-60分 → 黄色预警，显示风险提示
> 60分  → 红色拦截，强烈建议暂缓
    ↓
用户确认 → ATK · placeOrder 执行下单
         → ATK · setStopLoss 自动附加止损
```

**拦截的典型场景：**
- FOMO 追涨：价格暴涨后短时间内下单，情绪评分异常
- 逆势操作：趋势明显偏多时做空，资金费率为正
- 仓位过重：单笔超过总资金 20%
- 报复性加仓：刚刚亏损后立刻开反向仓

## OKX Agent Trade Kit 调用详情

交易教练深度集成 OKX ATK，以下是具体调用的工具：

| ATK 工具 | 调用场景 | 作用 |
|---------|---------|------|
| `getTrades` | 健康度评分 / 周报 | 读取当日 / 当周所有交易记录 |
| `getPositions` | 健康度评分 / 风险评分 | 读取当前持仓和杠杆率 |
| `getAccountStats` | 周报 KPI | 读取账户总体表现数据 |
| `getRiskLevel` | 下单前评分 | 获取账户当前风险等级 |
| `getMarketData` | 下单前评分 | 获取当前趋势和资金费率 |
| `placeOrder` | 下单执行 | 用户确认后执行下单 |
| `setStopLoss` | 下单执行 | 自动为每笔单附加止损 |
| `getTradeHistory` | 成就系统 / 周报 | 读取历史交易计算连击天数 |

**OpenClaw 的作用：**
- 实时采集 X 平台市场情绪，输入情绪稳定度评分
- 评分触发预警时主动推送通知
- 每日交易行为记录，用于计算成就和连击
